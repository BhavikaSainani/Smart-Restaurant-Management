# BiteWise - Smart Restaurant Management

Welcome to **BiteWise**, a comprehensive and smart restaurant management platform.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Prerequisites

- Node.js & npm installed

## Getting Started

Follow these steps to set up the local development environment:

```sh
# Step 1: Install the necessary dependencies
npm install

# Step 2: Start the development server
npm run dev
```

## Running Locally

After running `npm run dev`, your application will be available at [http://localhost:5173](http://localhost:5173).
